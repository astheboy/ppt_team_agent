학생용 질문 프레임워크 (5C)

04

C

1

Clear (명확성) - 구체적이고 명확한 질문

"뭐야?"가 아닌 "어떻게", "왜", "무엇을" 명확히 질문하기

예시

"광합성의 3단계 과정을 순서대로 설명해줘"

C

2

Context (맥락) - 배경 정보 제공

나의 수준, 학습 목적, 현재 상황을 AI에게 알려주기

예시

"중학생 수준에서", "과학 프로젝트를 위해"

C

3

Constraint (제약) - 조건과 범위 설정

답변의 길이, 형식, 방식 등을 구체적으로 지정하기

예시

"3문단으로", "표로 정리해서", "비유를 들어"

C

4

Chunk (단계화) - 복잡한 질문 나누기

어려운 주제는 작은 단계로 쪼개서 순차적으로 질문하기

예시

"먼저 개념 설명 → 다음 예시 → 마지막 적용"

C

5

Check (확인) - 결과 검증 및 후속 질문

답변을 이해했는지 확인하고, 추가 질문으로 심화하기

예시

"내가 이해한 게 맞는지 확인해줘", "더 자세히"
